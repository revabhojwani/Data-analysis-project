# Titanic Survival Analysis — Data Analytics Mini Project

A data analytics project analyzing the Titanic passenger dataset to identify factors that influenced survival, built as part of the Data Analytics course (CSS 2103) at Manipal Institute of Technology.

## About

This project explores the Titanic dataset to understand how gender, passenger class, age, and fare affected survival outcomes. It includes data preprocessing, exploratory data analysis, hypothesis testing, and predictive modeling using logistic regression.

## Key Findings
- Female passengers had significantly higher survival rates than male passengers
- First-class passengers survived at much higher rates than third-class passengers
- Higher fare paid was positively correlated with survival
- Logistic regression model achieved 75.4% accuracy

## Tech Stack
- **Language:** Python (Jupyter Notebook)
- **Data manipulation:** pandas, NumPy
- **Visualization:** Matplotlib, Seaborn
- **Statistics:** SciPy (Chi-Square test, t-tests)
- **Machine Learning:** Scikit-learn (Logistic Regression)

## Project Structure
- Data preprocessing — handled missing values, encoded categorical variables
- Exploratory Data Analysis — age distribution, fare distribution, survival by gender and class
- Hypothesis Testing — Chi-Square tests for gender and class vs survival
- Predictive Modeling — Logistic Regression with Age, Sex, Pclass, Fare as features

## Dataset
Kaggle Titanic: Machine Learning from Disaster
- 891 records, 12 attributes
